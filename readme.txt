Dusan Kovacic's Portfolio Website

This repository showcases the source code for my personal portfolio website, a platform where design meets development and user experience takes center stage. It serves as a comprehensive overview of my skills and experience as a graphic designer and front-end developer, where I seamlessly blend aesthetics with functionality to craft captivating digital experiences.

Key Features:

Responsive Design: The website adapts flawlessly across desktops, tablets, and mobile devices, ensuring an optimal viewing experience on any platform.
Interactive Elements: Subtle animations and hover effects bring the design to life, fostering an engaging and immersive user journey.
Clean and Maintainable Code: The codebase is meticulously structured and well-commented, prioritizing readability and facilitating future collaboration.
Accessibility-Focused Design: The website adheres to best practices for accessibility, guaranteeing an inclusive and smooth experience for all users.
Getting Started

To delve into the website locally, follow these straightforward instructions:

Clone the Repository: Using Git, execute the following command: git clone https://github.com/your-username/your-repository.git (replace your-username and your-repository with your actual details).
Navigate to the Project Directory: Use the command cd your-repository to enter the project directory.
Launch the Website: Open the index.html file in your preferred web browser to view the portfolio.
